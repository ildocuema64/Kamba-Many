# Guia de Atualização PWA

Para habilitar a funcionalidade PWA no cliente:

1.  Execute `git pull origin main` (ou baixe o código novo).
2.  Execute `npm install` (Novas dependências foram adicionadas).
3.  Execute `npm run build`.
4.  Reinicie o servidor com `npm start`.

O botão "Instalar App" aparecerá no canto inferior da tela.
